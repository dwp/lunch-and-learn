## Get Notified If you wish to be notified of new lunch and learn sessions, the best way at the moment is to watch the repo for this site for changes. ## Submit an idea ### I want to present something Great! Please ensure you have a room booked out before opening a pull request. ### I have something I'd like to learn We're always open to ideas, and calling out what you'd like to learn is a great start!
